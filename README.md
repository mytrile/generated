# generated

generated is simple library for detecting if a file is generated (CoffeeScript files, XCode project files, etc).

# Usage

~~~ go
package main

import (
    "github.com/mytrile/generated"
)

func main() {
    generated := generated.IsGenerated("/path/to/file") // true/false
}

~~~
